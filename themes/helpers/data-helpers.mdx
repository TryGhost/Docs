---
title: Data Helpers
description: Data helpers are used to output data from your site. Use this reference list to discover what each handlebars helper can do when building a custom Ghost theme.
---

| Tag                                                              | Description                                                          |
| ---------------------------------------------------------------- | -------------------------------------------------------------------- |
| [@config](/themes/helpers/config/)                          | Provides access to global data properties                            |
| [@custom](/themes/helpers/custom/)                          | Provides access to custom theme settings                             |
| [@page](/themes/helpers/page/)                              | Provides access to page settings                                     |
| [@site](/themes/helpers/site/)                              | Provides access to global settings                                   |
| [@member](/themes/members/#the-member-object)               | Provides access to member data                                       |
| [authors](/themes/helpers/authors/)                         | Outputs the post author(s)                                           |
| [comments](/themes/helpers/comments/)                       | Outputs Ghost's member-based commenting system                       |
| [content](/themes/helpers/content/)                         | Outputs the full post content as HTML                                |
| [date](/themes/helpers/date/)                               | Outputs the date in a format of your choosing                        |
| [excerpt](/themes/helpers/excerpt/)                         | Outputs the custom excerpt, or the post content with HTML stripped   |
| [facebook](/themes/helpers/facebook/)                       | Outputs the full URL to the Facebook profile from Settings           |
| [social\_url](/themes/helpers/social_url/)                  | Outputs the full URL to a social profile                             |
| [img\_url](/themes/helpers/img_url/)                        | Outputs the correctly calculated URL for the provided image property |
| [link](/themes/helpers/link/)                               | Creates links with dynamic classes                                   |
| [navigation](/themes/helpers/navigation/)                   | Helper which outputs formatted HTML for navigation links             |
| [post](/themes/helpers/post/)                               | More `object` than helper â€“ Contains all data for a specific post    |
| [price](/themes/helpers/price/)                             | Outputs a price with formatting options                              |
| [readable\_url](/themes/helpers/readable_url/)              | Returns a human-readable URL                                         |
| [recommendations](/themes/helpers/recommendations/)         | Outputs a list of recommended sites                                  |
| [tags](/themes/helpers/tags/)                               | Outputs the post tags                                                |
| [tiers](/themes/helpers/tiers/)                             | Outputs the post tier(s)                                             |
| [title](/themes/helpers/title/)                             | The post title, when inside the `post` scope                         |
| [total\_members](/themes/helpers/total_members/)            | Outputs the number of members, rounded and humanised                 |
| [total\_paid\_members](/themes/helpers/total_paid_members/) | Outputs the number of paying members, rounded and humanised          |
| [twitter](/themes/helpers/twitter/)                         | Outputs the full URL to the Twitter profile from Settings            |
| [url](/themes/helpers/url/)                                 | The post URL, when inside the `post` scope                           |
