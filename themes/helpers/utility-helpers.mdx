---
title: Utility Helpers
description: Utility helpers are used to perform minor, optional tasks. Use this reference list to discover what each handlebars helper can do when building a custom Ghost theme.
---

| Tag                                                                                                         | Description                                                                      |
| ----------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| [asset](/themes/helpers/asset/)                                                                        | Outputs cachable and cache-busting relative URLs to various asset types          |
| [block](/themes/helpers/block/)                                                                        | Used along with `{{contentFor}}` to pass data up and down the template hierarchy |
| [body\_class](/themes/helpers/body_class/)                                                             | Outputs dynamic CSS classes intended for the `<body>` tag                        |
| [concat](/themes/helpers/concat/)                                                                      | Concatenate and link multiple things together                                    |
| [encode](/themes/helpers/encode/)                                                                      | Encode text to be safely used in a URL                                           |
| [ghost\_head](/themes/helpers/ghost_head_foot/) / [ghost\_foot](/themes/helpers/ghost_head_foot/) | Outputs vital system information at the top and bottom of the document           |
| [link\_class](/themes/helpers/link_class/)                                                             | Add dynamic classes depending on the currently viewed page                       |
| [log](/themes/helpers/log/)                                                                            | In development mode, output data in the console                                  |
| [pagination](/themes/helpers/pagination/)                                                              | Helper which outputs formatted HTML for pagination links                         |
| [partials](/themes/helpers/partials/)                                                                  | Include chunks of reusable template code                                         |
| [plural](/themes/helpers/plural/)                                                                      | Output different text based on a given input                                     |
| [post\_class](/themes/helpers/post_class/)                                                             | Outputs classes intended for your post container                                 |
| [prev\_post](/themes/helpers/prev_next_post/) / [next\_post](/themes/helpers/prev_next_post/)     | Within the `post` scope, returns the URL to the previous or next post            |
| [reading\_time](/themes/helpers/reading_time/)                                                         | Renders the estimated reading time for a post                                    |
| [search](/themes/helpers/search/)                                                                      | Output a working, pre-styled search button & icon                                |
| [translate](/themes/helpers/translate/)                                                                | Output text in your site language (the backbone of i18n)                         |
